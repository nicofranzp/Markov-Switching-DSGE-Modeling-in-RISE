# FAQ

**Q: How do I install RISE?**
A: Use the vendored toolbox in `vendor/RISE_toolbox`, add it to the MATLAB path, and run `rise_startup()`.

**Q: How do I specify priors?**
A: Use a struct or helper function, and pass to `estimate` with the 'estim_priors' option.

**Q: How do I get help?**
A: See the documentation, examples, or email the maintainer.
